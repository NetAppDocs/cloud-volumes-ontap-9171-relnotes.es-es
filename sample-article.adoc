= El título de la página va aquí
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Usa esta plantilla para artículos, temas, o como los llames. El archivo index.adoc no tiene las palabras clave correctas para una página de artículo y usarlo como plantilla puede causar problemas de publicación después.


NOTE: Elimina este archivo antes de la publicación inicial, ya que el archivo se publicará aunque no esté vinculado a la barra lateral. Después de crear unos cuantos artículos, puedes usarlos como punto de partida en su lugar.

Si tus artículos van a estar en una subcarpeta de contenido, tendrás que hacer dos ajustes a la notación mostrada arriba:

* `:imagesdir:../media/`
* `permalink: folder-name/sample-article.html`
